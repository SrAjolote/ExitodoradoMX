## 📋 Descripción

ExitodoradoMX es una plataforma completa de rifas desarrollada en PHP que permite a los usuarios:

- 🎟️ **Comprar y vender boletos** para distintas rifas
- 📊 **Gestionar rifas activas** con panel de administración
- 💰 **Administrar transacciones** de manera segura
- 👥 **Gestionar usuarios** y permisos
- 📱 **Interfaz responsive** para todos los dispositivos

Este proyecto demuestra habilidades avanzadas en desarrollo backend, manejo de bases de datos y lógica de administración de transacciones.

---

## 🗃️ Estructura de la Base de Datos

La plataforma gestiona las siguientes entidades principales:

- **Usuarios**: Registro y autenticación de usuarios
- **Rifas**: Información de cada rifa disponible
- **Boletos**: Control de boletos disponibles y vendidos
- **Transacciones**: Historial de compras y ventas
- **Pagos**: Registro de métodos y estados de pago

---

## 🚀 Características Técnicas

- Arquitectura MVC personalizada
- Validación de formularios tanto en frontend como backend
- Protección contra inyecciones SQL y ataques XSS
- Sistema de paginación para listados extensos
- API RESTful para futuras integraciones
- Optimización de consultas a base de datos

## 📦 Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/tuusuario/exitodoradomx.git
```

2. Importar la base de datos:
```bash
mysql -u usuario -p exitodoradomx < database/exitodoradomx.sql
```

3. Configurar las variables de entorno en `config/database.php`

4. Acceder al sistema desde el navegador
